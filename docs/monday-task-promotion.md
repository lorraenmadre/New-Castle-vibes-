# Monday.com Task Promotion

New Castle promotes document-based work into Monday.com only when the work is clear enough to execute.

Not every document becomes a task.

Some documents need only storage.
Some need summary.
Some need human review.
Some need a deadline, owner, and next action.

## Purpose

The Monday.com promotion rule keeps New Castle from flooding the execution board with vague anxiety disguised as productivity.

A document becomes a Monday.com item only when it has a specific action to track.

## Promotion Requirements

A document can become a Monday.com task when it has:

1. A clear next action.
2. A WISH WELL house assignment.
3. A document summary or reference link.
4. A decision, deadline, or review need.
5. A destination folder or source location.

## Do Not Promote When

Do not create a Monday.com task if:

- The document only needs storage.
- The document has not been reviewed enough to identify an action.
- The task is emotionally loud but practically unclear.
- The file is a duplicate.
- The document belongs in archive.
- More information is needed before action.

## Suggested Columns

A New Castle task board should include:

- Status
- WISH WELL House
- Document Type
- Priority
- Due Date
- Review Needed
- Source Link
- Drive Link
- Next Action
- Notes
- Owner

## Example

Document: insurance claim statement

New Castle extracts:

- Claim number
- Date of event
- Agent name
- People involved
- Summary of incident
- Needed action

Monday.com item:

- Group: Legal / Admin / New Castle
- Status: Ready
- WISH WELL House: House 8
- Due Date: claim deadline if available
- Next Action: Review and submit statement

## Automation Goal

Future automation should:

1. Read New Castle routed document output.
2. Check whether task promotion requirements are met.
3. Create Monday.com item in the correct group.
4. Attach or link the Drive document.
5. Link back to New Castle summary.
6. Update status when reviewed or completed.

## Outcome

Monday.com should receive only actionable work.

New Castle handles document intake and clarity.
Monday.com handles execution.
That division keeps the system from becoming one giant over-caffeinated checklist swamp.
